/etc/nginx/vhost#

报错
nginx: [emerg] host not found in upstream “music.test.cn” in /usr/local/nginx/conf/vhost/yq.nginx.com.conf:19 这个错误。
解决
vim /etc/hosts

修改hosts文件，在hosts文件里面加上一句

127.0.0.1 localhost.localdomain music.test.cn
